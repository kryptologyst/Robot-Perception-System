## Installation Notes

### CUDA Support (Optional)
For GPU acceleration, install PyTorch with CUDA support:
```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
```

### Apple Silicon Support
For Apple Silicon Macs, use MPS backend:
```bash
pip install torch torchvision
```

### ROS 2 Integration (Optional)
If using ROS 2 integration, install ROS 2 Humble or later:
```bash
# Ubuntu
sudo apt install ros-humble-desktop
pip install rclpy
```

## Troubleshooting

### Common Issues

1. **OpenCV Installation**: If opencv-python fails, try:
   ```bash
   pip install opencv-python-headless
   ```

2. **Open3D Installation**: On some systems, you may need:
   ```bash
   pip install open3d --no-cache-dir
   ```

3. **Matplotlib Backend**: For headless systems:
   ```bash
   export MPLBACKEND=Agg
   ```

### Platform-Specific Notes

#### Windows
- Use Visual Studio Build Tools for compilation
- Consider using conda for easier dependency management

#### macOS
- May need to install Xcode command line tools
- Use Homebrew for system dependencies

#### Linux
- Install development headers: `sudo apt install python3-dev`
- For OpenCV: `sudo apt install libopencv-dev`
